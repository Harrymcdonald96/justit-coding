HTML Elements Summary

Element Description Example
Header Used for headings within the document. HTML offers six levels of headings, from h1 (the largest) to h6 (the smallest). <h1>This is a Header</h1>
Paragraph Defines a block of text as a paragraph. <p>This is a paragraph.</p>
Anchor Creates a hyperlink to other web pages, files, locations within the same page, or any URL. <a href="https://www.example.com">Visit Example.com</a>
Image Embeds an image into the document. <img src="image.jpg" alt="Description of Image">
List Used to list items. Can be ordered (ol) or unordered (ul). <ul><li>Item 1</li><li>Item 2</li></ul>
Table Used to display tabular data. <table><tr><td>Data 1</td><td>Data 2</td></tr></table>

For more detailed information on HTML, visit MDN Web Docs - HTML.
